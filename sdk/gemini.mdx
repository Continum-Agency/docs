## Overview

The Continum SDK provides seamless integration with Google's Gemini models, offering powerful multimodal capabilities and efficient processing.

## Supported Models

| Model Name | API Model ID | Capabilities |
|------------|--------------|--------------|
| `gemini_2_5_pro` | `gemini-2.5-pro` | Most capable, multimodal |
| `gemini_2_5_flash` | `gemini-2.5-flash` | Fast variant, multimodal |
| `gemini_2_0_flash` | `gemini-2.0-flash` | Previous gen, fast |
| `gemini_1_5_pro` | `gemini-1.5-pro` | Legacy, multimodal |

## Basic Usage

```typescript
import { Continum } from '@continum/sdk';

const continum = new Continum({
  continumKey: process.env.CONTINUM_KEY,
  apiKeys: {
    gemini: process.env.GEMINI_API_KEY
  },
  defaultSandbox: 'your-sandbox-slug'
});

const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [
    { role: 'user', content: 'What is the capital of France?' }
  ]
});

console.log(response.content);
```

## Multimodal Support

Gemini excels at processing multiple types of content simultaneously:

### Text + Images

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{
    role: 'user',

      { 
        type: 'image_url',
        image_url: {
          url: 'https://example.com/image.jpg'

  }]
});
```

## Function Calling

Gemini supports function calling for tool use:

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [
    { role: 'user', content: 'What is the weather in Tokyo?' }
  ],
  tools: [
    {
      type: 'function',
      function: {
        name: 'get_weather',
        description: 'Get current weather for a location',
        parameters: {
          type: 'object',
          properties: {
            location: {
              type: 'string',
              description: 'City name'
            },
            unit: {
              type: 'string',
              enum: ['celsius', 'fahrenheit']
            }
          },
          required: ['location']
        }
      }
    }
  ]
});
```

## Code Execution

Gemini can execute code and return results:

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [
    { role: 'user', content: 'Calculate the factorial of 10 using Python.' }
  ],
  tools: [
    {
      type: 'code_execution'
    }
  ]
});
```

## Advanced Options

### Temperature Control

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{ role: 'user', content: 'Be creative!' }],
  temperature: 0.9,  // 0.0 to 2.0
```

### Max Tokens

```typescript
const response = await continum.llm.gemini.gemini_2_5_flash.chat({
  messages: [{ role: 'user', content: 'Summarize briefly.' }],

```

### Safety Settings

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{ role: 'user', content: 'Tell me about...' }],

    {
      category: 'HARM_CATEGORY_HATE_SPEECH',

```

## Streaming Responses

Stream responses for real-time output:

```typescript
const stream = await continum.llm.gemini.gemini_2_5_flash.chat({
  messages: [
    { role: 'user', content: 'Write a long story.' }
  ],
  stream: true
});

for await (const chunk of stream) {
  process.stdout.write(chunk.content);
}
```

## Token Usage

Track token consumption:

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log('Input tokens:', response.promptTokens);
console.log('Output tokens:', response.outputTokens);
console.log('Total tokens:', response.promptTokens + response.outputTokens);
```

## Error Handling

```typescript
try {
  const response = await continum.llm.gemini.gemini_2_5_pro.chat({
    messages: [{ role: 'user', content: 'Hello' }]
  });
} catch (error) {
  if (error.message.includes('quota')) {
    console.error('API quota exceeded');
  } else if (error.message.includes('invalid_api_key')) {
    console.error('Invalid Gemini API key');
  } else if (error.message.includes('safety')) {
    console.error('Content blocked by safety filters');
  } else {
    console.error('Error:', error.message);
  }
}
```

## Model Selection Guide

### When to Use Each Model

**Gemini 2.5 Pro** - Most capable
- Complex multimodal tasks
- Long-form content generation
- Video and audio analysis
- Code generation and execution

**Gemini 2.5 Flash** - Fast and efficient
- Quick responses needed
- High-volume processing
- Simple multimodal tasks
- Cost-sensitive applications

**Gemini 2.0 Flash** - Previous generation
- Legacy applications
- Stable API requirements
- Budget constraints

## Best Practices

### 1. Leverage Multimodal Capabilities

```typescript
// Combine text, images, and context
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'Analyze this product image and suggest improvements.' },
      { type: 'image_url', image_url: { url: 'https://example.com/product.jpg' } },
      { type: 'text', text: 'Focus on design, usability, and market appeal.' }
    ]
  }]
});
```

### 2. Use Flash for Speed

```typescript
// For simple tasks, use Flash (faster and cheaper)
const quick = await continum.llm.gemini.gemini_2_5_flash.chat({
  messages: [{ role: 'user', content: 'Classify this text as positive or negative.' }]
});

// For complex tasks, use Pro
const detailed = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{ role: 'user', content: 'Analyze this video and create a detailed report.' }]
});
```

### 3. Handle Safety Filters

```typescript
const response = await continum.llm.gemini.gemini_2_5_pro.chat({
  messages: [{ role: 'user', content: 'Discuss sensitive topic...' }],
  safety_settings: [
    {
      category: 'HARM_CATEGORY_DANGEROUS_CONTENT',
      threshold: 'BLOCK_ONLY_HIGH'  // More permissive
    }
  ]
});
```
