## Overview

The Continum SDK provides seamless integration with Anthropic's Claude models, including Opus, Sonnet, and Haiku variants with extended thinking capabilities.

## Supported Models

| Model Name | API Model ID | Capabilities |
|------------|--------------|--------------|
| `opus_4_6` | `claude-opus-4-6` | Most capable, extended thinking |
| `sonnet_4_6` | `claude-sonnet-4-6` | Balanced, extended thinking |
| `sonnet_4` | `claude-sonnet-4-5` | Balanced performance |
| `haiku_4_5` | `claude-haiku-4-5-20251001` | Fast and efficient |
| `haiku_3_5` | `claude-3-5-haiku-20241022` | Previous gen, fast |
| `sonnet_3_7` | `claude-sonnet-3-7-20250219` | Previous gen, balanced |

<Note>
  **Legacy Format**: `claude_3_5_sonnet` is also supported (v0.0.4+) but the family name format above is recommended.
</Note>

## Basic Usage

```typescript
import { Continum } from '@continum/sdk';

const continum = new Continum({
  continumKey: process.env.CONTINUM_KEY,
  apiKeys: {
    anthropic: process.env.ANTHROPIC_API_KEY
  },
  defaultSandbox: 'your-sandbox-slug'
});

const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [
    { role: 'user', content: 'What is the capital of France?' }
  ]
});

console.log(response.content);
```

<Tip>
  You can use either `continum.llm.anthropic` or `continum.llm.claude` - they're aliases for the same thing.
</Tip>

## Extended Thinking

Claude Opus and Sonnet 4 models support extended thinking for complex reasoning:

```typescript
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [
    { role: 'user', content: 'Solve this complex problem step by step...' }
  ]
});

// Access the thinking process
console.log('Thinking:', response.thinkingBlock);
console.log('Answer:', response.content);
```

The SDK automatically enables thinking for capable models with a budget of 1000 tokens. If the user provides a `maxTokens` value that's too low (≤ 1000), the SDK automatically adjusts it to 3048 (1000 + 2048).

<Note>
  **Important**: When using thinking models, `max_tokens` must be greater than the thinking budget (1000). The SDK will automatically adjust if needed, but you can set a custom `maxTokens` of at least 1001.
</Note>

### Custom Max Tokens with Thinking

```typescript
// ✅ Correct - max_tokens > thinking budget
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [{ role: 'user', content: 'Complex task...' }],
  maxTokens: 5000  // Must be > 1000
});

// ✅ Auto-adjusted - SDK will set to 3048
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [{ role: 'user', content: 'Complex task...' }],
  maxTokens: 500  // Too low, SDK adjusts to 3048
});
```

## System Prompts

Claude uses system prompts differently than other models:

```typescript
const response = await continum.llm.anthropic.sonnet_4_6.chat({
  messages: [
    { role: 'user', content: 'Tell me about Paris.' }
  ],
  systemPrompt: 'You are a helpful travel guide assistant.'
});
```

<Note>
  Unlike OpenAI, Claude's system prompt is passed as a separate parameter, not as a message.
</Note>

## Vision Support

Claude models support image inputs:

```typescript
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'What is in this image?' },
      { 
        type: 'image',
        source: {
          type: 'url',
          url: 'https://example.com/image.jpg'
        }
      }
    ]
  }]
});
```

You can also use base64-encoded images:

```typescript
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'Describe this image.' },
      { 
        type: 'image',
        source: {
          type: 'base64',
          media_type: 'image/jpeg',
          data: 'base64_encoded_image_data_here'
        }
      }
    ]
  }]
});
```

## Tool Use (Function Calling)

Claude supports tool use for function calling:

```typescript
const response = await continum.llm.anthropic.sonnet_4_6.chat({
  messages: [
    { role: 'user', content: 'What is the weather in San Francisco?' }
  ],
  tools: [
    {
      name: 'get_weather',
      description: 'Get the current weather in a location',
      input_schema: {
        type: 'object',
        properties: {
          location: {
            type: 'string',
            description: 'The city and state, e.g. San Francisco, CA'
          },
          unit: {
            type: 'string',
            enum: ['celsius', 'fahrenheit'],
            description: 'Temperature unit'
          }
        },
        required: ['location']
      }
    }
  ]
});

// Check if Claude wants to use a tool
if (response.raw.stop_reason === 'tool_use') {
  const toolUse = response.raw.content.find(block => block.type === 'tool_use');
  console.log('Tool:', toolUse.name);
  console.log('Input:', toolUse.input);
}
```

## Multi-turn Conversations

Claude excels at multi-turn conversations:

```typescript
const messages = [
  { role: 'user', content: 'Hi, I need help planning a trip.' },
  { role: 'assistant', content: 'I\'d be happy to help! Where would you like to go?' },
  { role: 'user', content: 'Paris, France for 5 days.' }
];

const response = await continum.llm.anthropic.sonnet_4_6.chat({
  messages,
  systemPrompt: 'You are a helpful travel planning assistant.'
});
```

## Advanced Options

### Temperature Control

```typescript
const response = await continum.llm.anthropic.haiku_4_5.chat({
  messages: [{ role: 'user', content: 'Be creative!' }],
  temperature: 0.9  // 0.0 to 1.0 (higher = more creative)
});
```

<Note>
  Temperature is not available for thinking models (Opus 4, Sonnet 4) as they use extended thinking instead.
</Note>

### Max Tokens

```typescript
const response = await continum.llm.anthropic.sonnet_4_6.chat({
  messages: [{ role: 'user', content: 'Summarize briefly.' }],
  maxTokens: 1000  // Limit response length
});
```

### Stop Sequences

```typescript
const response = await continum.llm.anthropic.haiku_4_5.chat({
  messages: [{ role: 'user', content: 'List 5 items.' }],
  stop_sequences: ['\n\n', 'END']
});
```

## Token Usage

Track token consumption:

```typescript
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log('Input tokens:', response.promptTokens);
console.log('Output tokens:', response.outputTokens);
console.log('Total tokens:', response.promptTokens + response.outputTokens);
```

## Error Handling

```typescript
try {
  const response = await continum.llm.anthropic.opus_4_6.chat({
    messages: [{ role: 'user', content: 'Hello' }]
  });
} catch (error) {
  if (error.message.includes('rate_limit')) {
    console.error('Rate limit exceeded');
  } else if (error.message.includes('invalid_api_key')) {
    console.error('Invalid Anthropic API key');
  } else if (error.message.includes('overloaded')) {
    console.error('Anthropic API is overloaded, retry later');
  } else {
    console.error('Error:', error.message);
  }
}
```

## Model Selection Guide

### When to Use Each Model

**Opus 4.6** - Most capable
- Complex reasoning tasks
- Long-form content generation
- Code review and analysis
- Research and analysis

**Sonnet 4.6** - Balanced
- General-purpose tasks
- Customer support
- Content moderation
- Data extraction

**Haiku 4.5** - Fast and efficient
- Simple queries
- Classification tasks
- Quick summaries
- High-volume processing

## Best Practices

### 1. Use System Prompts Effectively

```typescript
const response = await continum.llm.anthropic.sonnet_4_6.chat({
  messages: [
    { role: 'user', content: 'Review this code...' }
  ],
  systemPrompt: `You are an expert code reviewer. Focus on:
- Security vulnerabilities
- Performance issues
- Best practices
- Code clarity`
});
```

### 2. Leverage Extended Thinking

```typescript
// For complex problems, use thinking models
const response = await continum.llm.anthropic.opus_4_6.chat({
  messages: [
    { role: 'user', content: 'Design a scalable architecture for...' }
  ]
});

// Review the thinking process
console.log('Reasoning:', response.thinkingBlock);
```

### 3. Choose the Right Model

```typescript
// Simple task - use Haiku (fast and cheap)
const simple = await continum.llm.anthropic.haiku_4_5.chat({
  messages: [{ role: 'user', content: 'Classify this email as spam or not.' }]
});

// Complex task - use Opus (most capable)
const complex = await continum.llm.anthropic.opus_4_6.chat({
  messages: [{ role: 'user', content: 'Analyze this legal document...' }]
});
```
